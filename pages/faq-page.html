GameScanner Frequently Asked Questions

Find answers to common questions about using GameScanner. Refresh your knowledge or troubleshoot issues with our comprehensive FAQ.

Why does GameScanner show an empty square?

GameScanner might show an empty square temporarily because:

  • It's gathering fresh data from multiple sources
  • Your connection might be slightly slow
  • We're processing complex game information

Solution refresh the page if it takes more than 30 seconds.

What data does GameScanner display?

GameScanner provides comprehensive game information including:

  • Game Title: The official name as listed on Steam
  • Release Date: When the game was officially launched
  • Current Price: Including any active discounts or sales
  • Genres: Game categories and types
  • Platforms: Available systems (Windows, Mac, Linux)
  • Store Links: Direct access to:
    • Steam store page
    • Official game website
    • GOG store (when available)
    • Epic Games Store (when available)

Note: The availability of information may vary depending on the game and its presence on different platforms.

How does GameScanner handle pricing data?

Our pricing information comes directly from the Steam API

  • Each time you visit a Twitch page containing a game title, GameScanner automatically fetches the current pricing data directly from Steam's official API
  • This ensures you always see the most up-to-date pricing, including any ongoing sales or discounts
  • For games available on multiple platforms, we prioritize Steam pricing data for consistency

Note: If a game isn't available on Steam or the API request fails, we'll indicate "Price not available"

Where does the game information come from?

We aggregate data from multiple trusted sources:

  • Official publisher APIs
  • IGDB (Internet Game Database)
  • Steam Web API

Note: If a game cannot be found, this will be indiciated by the GameScanner displaying "Game Not Found"

How do I report bugs or issues?

Please use our dedicated bug reporting system:

  1. Click the bug icon in the extension toolbar
  2. Fill out the detailed form
  3. Include screenshots if possible
  4. Submit your report

We typically respond within 48 hours.

Does GameScanner work offline?

GameScanner requires an internet connection to:

  • Fetch live pricing data
  • Update game information
  • Verify store availability
Why isn't a price showing for some games?

A price might not appear for several reasons:

  • The game isn't available on Steam
  • For some games the steam API does not contain the price.
  • The game has been removed from the Steam store
  • Temporary API connection issues

If you believe this is an error, please report it using our bug reporting system.

How do I disable or remove GameScanner?

To manage GameScanner:

  • Temporary disable: Click the extension icon and use the toggle switch
  • Remove completely: Right-click the extension icon and select "Remove from Chrome"
  • You can also manage extensions through chrome://extensions

Note: Disabling the extension will not delete any settings or data.

What browsers are supported?

Currently, GameScanner is available for:

  • Google Chrome
  • Microsoft Edge
  • Other Chromium-based browsers (like Brave or Opera)

Note: Support for Firefox and other browsers may be added in future updates.
